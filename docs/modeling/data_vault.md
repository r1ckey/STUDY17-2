# Data Vault 2.0
### 1. 【エンジニアの定義】Professional Definition
> **Data Vault 2.0**: 高度なデータエンジニアリング及び分散システム設計において、システムのスケール、信頼性、モデラビリティを担保するためのコア概念。
### 2. 【0ベース・深掘り解説】Gap Filling
Hub(ハブ), Link(リンク), Satellite(サテライト)の3構造で、超大規模かつ複数システム由来の「変化し続けるシステム」を柔軟に統合する最新モデリング。
