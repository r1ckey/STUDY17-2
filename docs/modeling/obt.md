# One Big Table (OBT)
### 1. 【エンジニアの定義】Professional Definition
> **One Big Table (OBT)**: 高度なデータエンジニアリング及び分散システム設計において、システムのスケール、信頼性、モデラビリティを担保するためのコア概念。
### 2. 【0ベース・深掘り解説】Gap Filling
JOINを避けるため、クエリ前に全てのDimensionを結合済みの超巨大「1枚の平べったい表」を作る現代DWHの力技最適化。
